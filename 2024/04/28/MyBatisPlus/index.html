1.常见注解

通过扫描实体类,并基于反射获取实体类信息作为数据库表信息

  • 类名转下划线作为表名
  • 名为id的字段作为主键
  • 变量名驼峰转下划线作为表字段名
常见注解 作用
@TableName 用来指定表名
@TableId 用来指定表中的主键字段信息
@TableField 用来指定表中的普通字段信息

1).@TableId

type 作用
AUTO 数据库自增长
INPUT 通过set方法自己输入
ASSIGN_ID(默认策略) 分配id,接口identifierGenerator的方法nextId来生成id,默认实现类为DefaultIdentifierGenerator雪花算法–类型(Long)

2).@TableField

常见使用场景

  • 成员变量名与数据库字段名不一致
  • 成员变量以is开头,且是Boolean值
  • 与数据库关键字冲突(要加``(esc下面那个)转义字符)
  • 成员变量不是数据库字段@TableField(exist = false)
